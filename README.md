## Live Chat Web APP - with Auth and Database ##

Full live-chat app based on needing somewhere to chat with fellow OU students on my module,with seperate rooms, user authentication using JWT and mongoDB 

All messages stored on mongoDB and brought back when app loads.

Tech used : 
    - Vanilla Javascript/HTML/CSS
    - MongoDB/Mongoose
    - WebSockets
    - JSON Web Tokens
    - Node.js/Express
    - GIT for version control
    - Heroku for deployment

My first "full stack" app
  

[Live Site - Here](https://webchat-unichat.herokuapp.com/)
